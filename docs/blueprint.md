# **App Name**: SmartCalc Hub

## Core Features:

- Mortgage Calculator: Calculates monthly mortgage payments based on principal, interest rate, and loan term.
- Compound Interest Calculator: Determines the future value of an investment with compound interest over time.
- Retirement Planner: Estimates savings needed and potential retirement income based on user inputs.
- Tax Calculator: Calculates estimated income tax based on specified income and tax rules.
- Currency Converter: Converts values between different currencies using real-time or recent exchange rates.
- Loan Interest Calculator: Computes the total interest paid and payment schedule for a given loan amount and interest rate.
- ROI Calculator: Measures the profitability of an investment by comparing gain to cost.
- YouTube Revenue Estimator: Provides an estimate of potential YouTube earnings based on views and other metrics.
- BMI Calculator: Calculates Body Mass Index (BMI) using height and weight to assess weight categories.
- Calorie Calculator: Estimates daily caloric needs based on activity level, age, sex, height, and weight.

## Style Guidelines:

- Primary color: Deep indigo (#3D29A3). This vibrant yet stable hue signifies trust, precision, and thoughtfulness, anchoring the design in reliability.
- Background color: Soft, light lavender-grey (#F1F0F5). Heavily desaturated from the primary hue, this color provides a clean, unobtrusive backdrop that enhances readability and maintains a sense of calm.
- Accent color: Clear blue (#3C73DD). This energetic and contrasting color is used for calls to action, important data highlights, and interactive elements, providing a clear visual cue without clashing with the primary.
- Headline and body text font: 'Inter' (sans-serif). Chosen for its modern, clean, and highly legible characteristics, 'Inter' ensures numerical data and explanatory text are clear and easy to read across all calculator interfaces.
- Use clear, intuitive line icons for each calculator category to provide instant visual recognition. Icons should be simple and universally understood.
- Each calculator should have a dedicated, minimalist interface with input fields prominently displayed. Results should be presented clearly with adequate spacing for legibility, ensuring an uncluttered user experience.
- Implement subtle transition animations when calculations are performed or when navigating between different calculator types. These smooth visual cues will enhance the interactive feel without distracting from the core functionality.
